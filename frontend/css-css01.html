# 使用伪元素实现超实用的图标库

伪元素是一个附加至选择器末的关键词，允许你对被选择元素的特定部分修改样式，用于向某些选择器设置特殊效果。伪元素主要有：

  • ::first-letter 第一个字母的样式
  • ::first-line 首行文字的样式
  • ::before 元素头部添加的修饰
  • ::after 元素尾部添加的修饰
  • ::placeholder input 的占位符样式
  • ::selection 被选中元素的样式
